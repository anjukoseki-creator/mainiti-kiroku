# 毎日記録

「昨日の自分を超えるためのAI成長ログ」

日記ではなく、毎日の **できたこと・学んだこと・明日やること** だけを30秒で記録し、
AIが長期的な成長を分析する個人用アプリ。比較対象は他人ではなく過去の自分。

## 使い方

```bash
npm install
npm run dev
```

データはブラウザの localStorage にのみ保存されます（完全ローカル）。
設定画面から JSON エクスポート / インポートが可能です。

## AIレビュー

- 設定画面で Claude API キーを設定すると、レビューボタンで自動分析
- 未設定でも「プロンプトをコピー」→ Claude に貼り付け → 結果を貼り戻して保存、で利用可能

## 技術

React + Vite + TypeScript / 依存最小・DBなし・完全ローカル動作
